Before You Submit
Print-Ready File Guide

We check every file before printing — but starting with the right setup saves time for everyone.

1
Resolution: 300 DPI minimum

Images must be 300 DPI at final print size. Screen-resolution images (72–96 DPI) will appear blurry when printed.

✓ We'll flag low-res images before printing
2
Color Mode: CMYK

Set your document to CMYK color mode, not RGB. RGB colors can shift significantly when converted to print.

✓ We convert RGB to CMYK if needed
3
Bleed: 0.125" on all sides

Extend background colors and images 1/8" beyond the trim edge to avoid white edges after cutting.

✓ Templates available on request
