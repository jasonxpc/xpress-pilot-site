다크 배경 헤드라인 전용 — Warm / Lime 강조색

⚠ 이 두 색상(+ Sky)은 어두운 배경 위의 짧은 강조 헤드라인에만 사용 — 예: Hero/Footer "Print Fast. Look Sharp. Stay Local.", 홈페이지 "Small run? We specialize in it.", By Industry 페이지 "One identity, every format." 흰색과 섞어서 단어별로 색상을 다르게 준다. 일반 본문/버튼/배지 등에는 절대 사용하지 말 것.

색상 사용 규칙 요약

소제목 (eyebrow label) — 모든 페이지, 모든 섹션 → var(--cyan) #0891B2 · 12px · uppercase · letter-spacing .1em
섹션 h2 제목 — 전체 사이트 → var(--slate) #1E2D3D (검정에 가까운 네이비)
Hero 내 소제목 — 어두운 배경 위 → var(--sky) #38BDF8 (밝은 파랑)
CTA 버튼 — "Get a Quote" 등 → var(--crimson) #DC2626
Hover / 활성 버튼 — Filter, Tab 등 활성 상태 → var(--cyan) · 배경 var(--cyan-tint)
Warm / Lime / Sky 혼합 강조 — 다크 배경의 짧은 강조 헤드라인에만 (Hero/Footer 슬로건, Small-run 섹션, By Industry 헤드라인). 일반 UI 요소엔 사용 금지
Nav 드롭다운 배지 — 카테고리별 고유 색상 (위 표 참고) · 사이언 금지 (hover와 혼동)
